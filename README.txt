Tanks
=====
Tank game written in Python using pygame.

Requirements
------------
Python 3 with pygame. <https://www.pygame.org/>

Installation
------------
1. Download the latest version of the game from:
  <https://github.com/spitko/tanks/archive/master.zip>
2. Unzip the downloaded file
3. Make sure you have Python 3 with pygame installed

Usage
-----
Run tanks.pyw with Python
